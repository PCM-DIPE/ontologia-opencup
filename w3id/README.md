# OpenCUP Ontology

## Permanent Identifier

`https://w3id.org/italia/onto/OpenCUP/`

## Description

The **OpenCUP Ontology** models the domain of the *Codice Unico di Progetto* (CUP) â€” the Unique Project Code for Italian public investment projects.

It is part of the **OntoPiA** network (`https://w3id.org/italia/onto/`) and the Italian national semantic data catalog [Schema.gov.it](https://schema.gov.it).
